# 🎬 Movie Mate

A TypeScript-based React app featuring React Hook Form with custom hooks, Node.js HTTP API integration, Material UI, and efficient state management using Context API, useReducer, useCallback, and useEffect.

---

## 🚀 Features
- 🔍 Search and explore movies  
- 📡 Integration with custom Node.js HTTP API  
- 📝 Form handling using React Hook Form + custom hooks  
- 🧠 State management using Context API and useReducer  
- ⚡ Performance optimization with useCallback  
- 🔄 Side-effect handling with useEffect  
- 🎨 Responsive UI using Material UI  

---

## 🛠 Tech Stack
- React.js + TypeScript  
- React Hook Form  
- Material UI (MUI)  
- Context API + useReducer  
- Node.js (HTTP Server)  
- Axios / Fetch API
